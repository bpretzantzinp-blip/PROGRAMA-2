Tarea 3 - Estructuras, Nodos y Lista Simple

Descripcion:

Este programa crea una lista simplemente enlazada utilizando estructuras y memoria dinamica.
Se crean 4 nodos dinamicamente usando new.
El usuario ingresa codigo, nombre y precio de cada producto.
Luego se recorre la lista con un puntero auxiliar para mostrar los datos.
Tambien se calcula el total acumulado de los precios.
Finalmente se libera la memoria correctamente usando delete.
